\chapter{Description}
\label{sec:description}

\section{Purpose}

This document describes the interface control for the Shelter (SHL) subsystem of the Long Wavelength Array (LWA). The SHL subsystem is responsible for monitoring and controlling the environmental conditions, power distribution, and safety systems within the station shelter.

\section{Scope}

This Interface Control Document (ICD) defines:
\begin{itemize}
    \item Physical interfaces to and from SHL
    \item Monitor and control (M\&C) interfaces
    \item Safety interfaces
    \item Command and response message formats
    \item Management Information Base (MIB) entries
\end{itemize}

\section{System Overview}

The SHL subsystem performs the following functions:
\begin{itemize}
    \item \textbf{Power Distribution}: Control and monitoring of rack-mounted PDUs and UPSs
    \item \textbf{Environmental Control}: Temperature monitoring and HVAC system interface
    \item \textbf{Weather Monitoring}: Integration with on-site weather station
    \item \textbf{Lightning Detection}: Interface with electric field mill for lightning alerts
    \item \textbf{Voltage Monitoring}: Detection of power flickers and outages
    \item \textbf{Safety Monitoring}: Smoke detection, water detection, and door status
\end{itemize}

\begin{siteNote}
The specific hardware configuration varies by LWA site. The number of PDUs, type of HVAC controller, and availability of environmental sensors depends on the site installation. Site-specific configurations are defined in the \texttt{defaults.json} configuration file.
\end{siteNote}
